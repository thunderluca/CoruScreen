Your browser is not supported.
CoruScreen does not support the following browser:
  • iOS / iPad Safari-based browsers
  • Firefox (Desktop and Android) from version 69
This browsers are not supported due to:
  • Security concerns that do not allow CoruScreen to retrieve device lists
  • Unreliable streaming experience, like continuos interruptions, background streaming not allowed, and so on.
CoruScreen recommends to use a Chromium-based desktop browser to stream without this issues.
